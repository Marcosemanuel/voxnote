# Segurança

## Reportar vulnerabilidades

Não publique detalhes de uma vulnerabilidade em issue pública. Envie um relatório privado aos mantenedores do repositório com:

- descrição do problema e impacto;
- passos mínimos para reproduzir;
- versão afetada;
- evidência sem anexar áudio, credenciais ou dados pessoais.

O Voxnote processa áudio localmente. Ainda assim, não inclua arquivos reais de usuários nos relatos ou pull requests.

## Escopo

O instalador e o aplicativo suportam Windows 10/11 x64. Dependências vulneráveis devem ser atualizadas por alteração revisável, com testes e registro no changelog.
